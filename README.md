# DEEP-LEARNING
Proyecto de aprendizaje automático para clasificar riesgo metabólico en pacientes usando exámenes clínicos relacionados con obesidad. Incluye limpieza de datos, construcción del diagnóstico mediante reglas médicas y entrenamiento de una red neuronal MLP para predecir el nivel de riesgo.
# Clasificación de Riesgo Metabólico con Redes Neuronales (MLP)

## Descripción

Este proyecto implementa un modelo de aprendizaje automático para clasificar el riesgo metabólico en pacientes a partir de resultados de exámenes clínicos relacionados con patologías asociadas a la obesidad. Se desarrolla un flujo completo que incluye limpieza de datos, construcción del diagnóstico mediante reglas médicas y entrenamiento de una red neuronal tipo MLP.

## Contexto académico

Este trabajo fue desarrollado como parte del curso de Deep Learning, con el objetivo de aplicar redes neuronales artificiales al análisis de datos clínicos reales y comprender el proceso completo de construcción de modelos predictivos.

## Objetivos del proyecto

* Preparar y limpiar el dataset clínico
* Identificar variables médicas relevantes
* Generar la etiqueta de riesgo metabólico
* Entrenar una red neuronal MLP
* Evaluar el desempeño del modelo

## Tecnologías utilizadas

* Python
* Pandas y NumPy
* Scikit-learn
* TensorFlow / Keras
* Jupyter Notebook

## Flujo de trabajo

1. Exploración del dataset
2. Limpieza y transformación de datos
3. Construcción del diagnóstico clínico
4. División en conjuntos de entrenamiento y prueba
5. Escalado de variables
6. Entrenamiento del modelo MLP
7. Evaluación del modelo

## Resultados esperados

El modelo busca aprender patrones clínicos a partir de valores de laboratorio para clasificar pacientes según su riesgo metabólico, aportando una aproximación computacional al análisis de información biomédica.
